# Just another Netflix Clone

Just another Netflix clone

## Getting Started

### Prerequisites

- Node or Web Server such as nginx / apache

### Installing

```bash
git clone https://github.com/ngochieu642/netflix-ui-clone.git netflix
cd netflix
npm i
npm run dev
```

The server will be up and running at port 5000
